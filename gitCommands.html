Git Commands

git restore--staged

Tar bort filer från staging men låter förändringar som gjort vara ifred

git restore

Hämtar tillbaka gamla filer eller kastar ej-inlagda filer i det lokala förrådet

git config--global user.name

används för att sätta ett fast användarnamn

git config--global user.email

används för att sätta sin email

git config --list

Hämtar alla inställningar git kan hitta vid det tillfället

git Branch

Lägger upp, skapar eller tar bort grenar i projektet

git config

Lägger in lokala och globala inställningar på projektet

git init

skapar en nytt tomt git repository eller återinitierar en redan existerande

git clone

kopierar och skapar en ny kopia av ett redan existerande repository

git status

visar vad för värden filerna har vid det tillfället

git status-s

visar mindre värden som filen har vid det tillfället

git status-long

visar värden som filen har vid det tillfället, visas som standard

git status-b

visar värden på grenen och värden på filen förkortat

git add

lägger till innehållet i filen till indexet

git add-n

visar endast filen/filer ifall de finns med och/eller blir ignorerade

git add-f

tillåter att filen läggs in även ifall den egentligen skulle bli ignorerad
