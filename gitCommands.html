Git Commands

git fetch

Hämtar ner alla grenar from den git remote

git push

Laddar upp innehållet från alla "commits" man gjort till Git

git pull

Laddar ner och slår ihop förändringar från Git med aktuell kod

git rebase 

rebase, flyttar man alla ändringar som sparat i en gren och laddar upp det i en annan gren
