Git Commands

git fetch

Hämtar ner alla grenar from den git remote

git push

Laddar upp innehållet från alla "commits" man gjort till Git

git pull

Laddar ner och slår ihop förändringar från Git med aktuell kod

git rebase

rebase, flyttar man alla ändringar som sparat i en gren och laddar upp det i en annan gren

git reset

rensar saging området, skriver om working tree till det specificerade commiten

git stash

sparar modifierad och staged ändringar

git stash list

listar stack-ordning av stashed fil ändringar

git stash pop

skriver working från toppen av stash stack

git stash drop

förkasta ändringarna från toppen av stash stack

git notes

Lägg till eller inspektera objekts noteringar
